# React Project Management App

A practice application built to demonstrate core React concepts including state management, API communication, and routing.

## Purpose

This app is designed to practice and showcase fundamental React concepts:

- **API Communication**: Making HTTP requests to external APIs
- **React Router**: Client-side routing and navigation
- **State Management**: Using `useState` for local component state
- **Side Effects**: Using `useEffect` for data fetching and lifecycle management
- **Component Architecture**: Building reusable components
- **Form Handling**: Creating and editing data through forms

## Features

- View a list of projects
- View project details
- Create new projects
- Edit existing projects
- Delete projects
- Add tasks to projects


## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local development URL (usually `http://localhost:5173`)

## Project Structure

- `src/components/` - Reusable React components
- `src/pages/` - Page components for different routes
- `src/config/` - API configuration
- `public/` - Static assets
